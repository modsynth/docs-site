# Backend Modules

## Available Modules

- **auth-module** - JWT + OAuth2.0 authentication
- **db-module** - Database abstraction with GORM
- **cache-module** - Redis cache client
- **logging-module** - Structured logging with Zap
- **api-gateway** - API Gateway with Gin
- **messaging-module** - RabbitMQ/Kafka abstraction
- **file-storage-module** - S3/MinIO storage
- **notification-module** - Email/SMS/Push notifications
