# Introduction

Welcome to Modsynth documentation!

Modsynth is a modular architecture framework for building modern applications.

## Quick Start

Install modules you need:

```bash
# Backend
go get github.com/modsynth/auth-module
go get github.com/modsynth/db-module

# Frontend
npm install @modsynth/ui-components
npm install @modsynth/api-client
```

## Features

- 19+ ready-to-use modules
- Multi-language support (Go, TypeScript)
- Comprehensive documentation
- Active development
